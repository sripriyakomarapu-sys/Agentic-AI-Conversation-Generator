# Agentic AI Conversation Generator

## Project Overview

The Agentic AI Conversation Generator is a web-based tool that creates high-quality prompts for simulating realistic agentic AI workflows.

Users can define a scenario, goal, agent persona, available tools, tone, and number of workflow steps. The tool then generates a structured prompt that can be used in ChatGPT, Claude, Gemini, or Perplexity.

## Features

- Scenario-based prompt generation
- Agent persona selection
- Tool selection
- Tone customization
- Multi-step workflow generation
- Copy prompt functionality
- Documentation section
- Sample run demonstration

## Why These Input Fields Were Chosen

Scenario: Defines context.

Goal: Defines the objective.

Agent Persona: Maintains consistency.

Available Tools: Creates realistic simulations.

Tone: Controls communication style.

Number of Steps: Controls workflow complexity.

## Prompt Structure

1. Agent Persona
2. Goal
3. Available Tools
4. Workflow Steps
5. Thought Process
6. Action Taken
7. Tool Used
8. Result
9. Next Decision

## Recommended AI Tools

- ChatGPT
- Claude
- Gemini
- Perplexity

## Sample Run

Scenario: Project Planning

Goal: Launch Company Website

Persona: Project Manager

Tools: Email, Calendar, Documents

Tone: Professional

Steps: 5

## How To Use

1. Enter scenario and goal.
2. Select persona.
3. Choose tools.
4. Select tone and steps.
5. Click Generate Prompt.
6. Copy and use in an AI tool.
